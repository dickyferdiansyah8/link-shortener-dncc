# 🚀 DNCC Link Shortener - Docker Edition

Project ini adalah tugas implementasi Cloud & DevOps untuk memendekkan URL menggunakan Laravel 13 dan Docker.

## 🛠️ Tech Stack
- **Framework:** Laravel 13 (PHP 8.4)
- **Web Server:** Nginx (Alpine)
- **Container:** Docker & Docker Compose
- **Database:** SQLite

## ⚡ Cara Menjalankan (Deployment)
Pastikan Anda sudah menginstal Docker di mesin Anda, lalu jalankan perintah berikut:

   ```bash
